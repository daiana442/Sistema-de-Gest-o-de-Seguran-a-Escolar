# Sistema de Gestão de Segurança Escolar

## Descrição
O Sistema de Gestão de Segurança Escolar é uma aplicação web desenvolvida para melhorar o controle e monitoramento da segurança no ambiente escolar.

O sistema permite registrar ocorrências, controlar entrada e saída de alunos e visitantes, emitir alertas e gerar relatórios gerenciais.

## Objetivo
Desenvolver um sistema informatizado para registrar e monitorar eventos relacionados à segurança escolar.

## Funcionalidades
- Controle de entrada e saída de alunos e visitantes
- Registro de ocorrências escolares
- Emissão de alertas
- Geração de relatórios
- Centralização das informações de segurança

## Tecnologias Utilizadas
- Sistema Web
- Banco de Dados Relacional
- Modelagem com dbdiagram.io
- Prototipação com Figma
